---

## 🚀 **What I’m Building — MPDO / Zimba Studio **
MPDO is a next‑generation engineering environment that brings together:

- **Modelica system modeling**  
- **FreeCAD parametric CAD generation**  
- **KiCad electronics design**  
- **FEMM/FMEAT motor & actuator analysis**  
- **Embedded C/C++ autocode generation**  
- **MiL/SiL/HiL verification workflows**  
- **AI‑assisted design and documentation**  
- **Distributed HPC simulation**  
- **Manufacturing process planning**  
- **PX4 & AgOpenGPS autonomy integration**

The goal:  
**A single open platform where engineers can go from natural‑language ideas → verified digital prototypes → manufacturable machines.**
